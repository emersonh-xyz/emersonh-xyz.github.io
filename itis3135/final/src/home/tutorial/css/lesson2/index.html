The Basics of CSS

Lesson 2: Colors

You can specify colors in CSS using different formats:

  • p{color: red}; This is a red paragraph.
  • p{color: #00ff00}; This is a green paragraph.
  • p{color: rgb(0, 0, 255)}; This is a blue paragraph.
